# Security Policy

## SDK Status
The Volt SDK is currently in an early MVP stage. It is designed to facilitate interaction with the Volt protocol smart contracts.

## Contract Risks
The Volt protocol smart contracts may be **unaudited**. We strongly advise users to verify contract addresses and source code before signing any transactions.

## Vulnerability Reporting
We take the security of our software seriously. If you discover a vulnerability, please report it via email:

**Email:** security@volt-protocol.example.com

Please include a detailed description of the issue and, if possible, steps to reproduce it.

## Responsible Disclosure
We ask that you:
- Do not publicly disclose the vulnerability until we have had a reasonable amount of time to address it.
- Do not attempt to exploit the vulnerability for any purpose.
- Provide sufficient technical details to help us investigate and remediate the issue.

Thank you for helping us keep Volt secure.
